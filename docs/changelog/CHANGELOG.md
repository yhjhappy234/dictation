## 版本 1.2.0

### 2026-04-18

#### 新功能
- **管理员审计日志查询**：
  - 审计日志列表页面（分页显示）
  - 按用户名、操作类型、时间范围搜索筛选
  - 日志详情查看（请求参数、执行结果、错误信息）
  - 操作统计（总日志数、成功/失败数）
  - 仅管理员可访问，自动权限校验
- **审计日志API接口**：
  - `GET /api/v1/audit-logs` - 分页获取审计日志列表
  - `GET /api/v1/audit-logs/search` - 条件搜索审计日志
  - `GET /api/v1/audit-logs/{id}` - 获取日志详情
  - `GET /api/v1/audit-logs/usernames` - 获取用户名列表（筛选）
  - `GET /api/v1/audit-logs/operations` - 获取操作类型列表（筛选）
  - `GET /api/v1/audit-logs/count` - 统计日志数量

## 版本 1.1.0

### 2026-04-18

#### 新功能
- **自动听写功能**：定时自动执行下一个词语，可设置间隔(1-60秒)，支持快捷键A
- **实时统计显示**：听写耗时、正确/错误数、进度百分比实时更新
- **预设内容分类优化**：
  - 常用词语：15个分类（水果、天气、颜色、动物、蔬菜等）
  - 成语分类：8个分类（数字成语、自然景色、人物描写等）
  - 古诗分类：8个分类（春季、夏季、秋季、山水、咏物等）
  - 古文分类：6个分类（修身篇、学习篇、自然篇等）
- **教材扩展**：
  - 初中语文教科书：七至九年级上下册
  - 高中语文教科书：高一至高三上下册
- **任务管理优化**：进行中任务显示正确/错误数和进度百分比
- **Flyway数据库版本管理**：支持数据库迁移和版本控制

#### 改进
- 预设内容选择界面支持多级分类选择（分类→年级→单元）
- 整合二年级下册单元文件为统一grade2-down.json
